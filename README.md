# DeltaStreamNet

[![CI](https://github.com/livedge/DeltaStreamNet/actions/workflows/ci.yml/badge.svg)](https://github.com/livedge/DeltaStreamNet/actions/workflows/ci.yml)
[![NuGet](https://img.shields.io/nuget/v/DeltaStreamNet.svg)](https://www.nuget.org/packages/DeltaStreamNet)

A .NET source-generated delta streaming library that transmits only changed properties over the wire instead of full object state, using Roslyn incremental generators.
